Данное мобильное приложение предназначено для прослушивания музыки и формирования собственной музыкальной библиотеки.

Для уcтановки мобильного приложения необходимо:
1. Скачать файл "LumiSound.apk" из папки "EXE" на мобильное устройство;
2. Подтвердить установку мобильного приложения;
3. Запустить приложение по ярлыку на рабочем столе мобильного устройства.

Для выполнения тестирования данного приложения на устройствах Android необходимы следующие технически требования: 
ӯ процессор 1,6 ГГц;
ӯ минимальный объем оперативной памяти – 1 ГБ;
ӯ операционная система Android 7.0 или более поздние версии.
